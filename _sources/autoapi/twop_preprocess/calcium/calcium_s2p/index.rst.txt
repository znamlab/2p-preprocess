.. py:function:: run_extraction(flz_session, project, session_name, conflicts, ops, delete_previous_run=False)

   Fetch data from flexilims and run suite2p with the provided settings

   Suite2p will generate a temporary folder, called suite2p where the initial binary
   are saved. It will save the actual output in the folder of the suite2p datasets.
   If conflict is overwrite, we re-run s2p.run but this function does not always redo
   everything is some files already exists. To start from a blank state, use
   delete_previous_run

   :param flz_session: flexilims session
   :type flz_session: Flexilims
   :param project: name of the project, determines save path
   :type project: str
   :param session_name: name of the session, used to find data on flexilims
   :type session_name: str
   :param conflicts: defines behavior if recordings have already been split
   :type conflicts: str
   :param ops: dictionary of suite2p settings
   :type ops: dict
   :param delete_previous_run: whether to delete previous runs. Default False
   :type delete_previous_run: bool

   :returns: object containing the generated dataset
   :rtype: Dataset
