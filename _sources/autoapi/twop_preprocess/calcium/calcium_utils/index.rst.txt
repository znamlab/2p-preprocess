.. py:function:: get_weights(ops)

   Get the weights for the suite2p detection.

   suite2p does not extract raw fluorescence traces, but rather uses a weighted sum of
   the raw fluorescence traces to detect ROIs. This function calculates the weights
   as in suite2p.detection.anatomical.select_rois (L161).
   For anatomical == 3 the weights are the intensity of each pixel relative to the
   1st and 99th percentile of the mean image + 0.1.

   :param ops: suite2p ops dictionary
   :type ops: dict

   :returns: weights for the suite2p detection
   :rtype: weights (numpy.ndarray)

.. py:function:: correct_neuropil_standard(F, Fneu, neucoeff, save_path=None)

   Applies standard neuropil correction: F = F - neucoeff * (Fneu - median(Fneu)).

   :param F: shape nrois x time, raw fluorescence trace for all rois
   :type F: numpy.ndarray
   :param Fneu: shape nrois x time, neuropil fluorescence trace for all
                rois
   :type Fneu: numpy.ndarray
   :param neucoeff: neuropil correction coefficient
   :type neucoeff: float
   :param save_path: path to save the neuropil corrected fluorescence trace. If None, will not save. Default None.
   :type save_path: str, optional

   :returns: shape nrois x time, neuropil corrected fluorescence trace
   :rtype: F_corrected (numpy.ndarray)
