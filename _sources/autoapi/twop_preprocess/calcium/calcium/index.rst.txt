.. py:function:: split_recordings(flz_session, suite2p_dataset, conflicts, base_name='suite2p_traces', extra_attributes=None)

   suite2p concatenates all the recordings in a given session into a single file.
   To facilitate downstream analyses, we cut them back into chunks and add them
   to flexilims as children of the corresponding recordings.

   :param flz_session: flexilims session
   :type flz_session: Flexilims
   :param suite2p_dataset: dataset containing concatenated recordings
                           to split
   :type suite2p_dataset: Dataset
   :param conflicts: defines behavior if recordings have already been split
   :type conflicts: str
   :param base_name: base name for the split datasets. Default
                     "suite2p_traces"
   :type base_name: str, optional
   :param extra_attributes: Extra attributes to add to the split datasets
                            on flexilims. Used to filter when looking for previously split datasets.
                            Default None
   :type extra_attributes: dict, optional
